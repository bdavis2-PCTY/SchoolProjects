I discovered my love for computer science when I first began writing basic scripts for online gaming servers when I was thirteen years old. This included writing plugins for Minecraft Bukkit  servers using Java, and add-ons for Garry's Mod servers Lua.

After getting into writing plugins and add-ons for games, I wanted to learn how to make websites. I began learning about web hosting and web development using HTML5, CSS3, and JavaScript/jQuery. I then got into the more advanced server-siding programing using PHP and a little of ASP.NET.

After learning about both game scripting and website development, I realized that I loved computer programming and it was something I could potentially do for a career. I then learned about desktop programming. I learned how to do basic stuff with C/C++ but I mainly focused on languages like Java and C#. I also learned how to make basic Android apps using the Android SDK.
